Citizens Standard Counterfactual

Running the replication

Requirements: Python 3.8+ with numpy and matplotlib.

```bash
# Reproduce all paper tables
python3 run\_all\_tables.py > all\_results.txt

# Regenerate figures
python3 mc\_plots.py

# Export the dataset as CSV
python3 build\_csv.py
```
1. Contents
/code/
Core Python scripts implementing the Model, the three issuance channels (K1, K2, K3), the dual‑circuit accounting structure, and the Monte Carlo bootstrap engine.

deterministic_engine.py — deterministic reconstruction of Mode B (1960–2055)

mc_engine.py — Monte Carlo bootstrap generator

authoritative_data.py — data loader and preprocessing

run_all_tables.py — produces all tables in the paper

mc_plots.py — produces all Monte Carlo figures

AUDIT.md — audit notes and verification steps

/data/
Historical datasets used for reconstruction and simulation.

citizens_standard_historical_data_1960_2025_v2.csv

citizens_standard_historical_data_1928_2025_full.csv

/figures/
All figures generated by the replication scripts, including:

distributional outcomes

percentile bands

below‑median probabilities

sensitivity analyses

/results/
Text outputs and intermediate files produced by the engines.

all_results.txt

intermediate CSVs and logs

2. How to Run
Install Python 3.10+

Install required packages:

Code
pip install -r requirements.txt
Run the deterministic reconstruction:

Code
python deterministic_engine.py
Generate all tables:

Code
python run_all_tables.py
Generate Monte Carlo figures:

Code
python mc_plots.py
All outputs will appear in /figures and /results.

3. Reproducibility Notes
All results in the paper are produced directly by the scripts in this directory.

No external data sources are required beyond the included CSV files.

Randomness in the Monte Carlo engine is fully controlled by fixed seeds for reproducibility.

The deterministic reconstruction is exact and produces identical results across environments.
