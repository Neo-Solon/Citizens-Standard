# The Citizens Standard — Paper 7: External Interoperability and the Common Anchor
### Replication package

This package reproduces every quantitative result in Paper 7.

## Contents
- `Citizens_Standard_External_Interoperability_Paper7.docx` — the paper
- The Technical Appendix (model spec, proofs, parameters, calibration, robustness, reproducibility) is now included at the end of `Citizens_Standard_External_Interoperability_Paper7.docx`.
- `code/` — all model scripts (see RESULTS_manifest.md for the claim→script map)
- `figures/` — publication figures (regenerated by `code/fig_paper7.py`)
- `outputs/` — captured stdout from every script
- `run_all.py` — one-command driver
- `requirements.txt` — pinned environment
- `RESULTS_manifest.md` — every claim → script → output line
- `LICENSE.txt`

## Reproduce
```
pip install -r requirements.txt
python run_all.py
```
All scripts are deterministic (fixed seeds). The domestic engine (`cs_engine.py`) reproduces
the framework's published launch figures to the dollar.

## Two classes of result (held to different standards)
1. **Domestic** (Sections 5–6, 9): computed on the CS issuance engine implemented from the
   published spec (Paper 1, §3.3, Appendix B) at real launch calibration (M2 = $22,366B; real
   growth 2%; K1 = 2.5% of GDP per capita). These reproduce the published figures exactly.
2. **External layer** (Section 7): EQUA's exact formula. Because EQUA is a *proposed* layer with
   no real-world instance, these are a **calibrated mechanism result** — robust across the
   empirically observed ranges of cross-country productivity growth and wage stickiness — not a
   forecast.

## Empirical calibration sources
- Wage stickiness (1–5yr range / Calvo durations): Card & Hyslop (1997); Barattieri, Basu &
  Gottschalk (2014); Daly & Hobijn (2014). Mean nominal-wage duration ≈ 12 months with downward
  rigidity. Theory: Taylor (1980); Calvo (1983).
- Cross-country productivity dispersion (0.5–3% gap): Feenstra, Inklaar & Timmer (2015, Penn
  World Table); World Bank, *Global Productivity* (Dieppe ed., 2021).
- Launch parameters (M2, GDP, population): public series (FRED M2SL; BEA; U.S. Census), 2025
  vintage, as anchored in the CS Macroeconomic Model paper (Neo-Solon, 2026a, App. A.1).
