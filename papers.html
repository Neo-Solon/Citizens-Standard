Replication & data

Built to be checked, not believed.

Eleven self-contained packages, one per empirical paper, each on genuine data: BLS CPI, the SF Fed inflation decomposition, SSA Trustees data, the 2022 Survey of Consumer Finances microdata, the Alaska Permanent Fund, and FRED monetary series. Every figure and table in the papers can be regenerated from source.

Each package runs the same way:

pip install -r requirements.txt
